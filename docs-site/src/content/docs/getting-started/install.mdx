The canonical install guide is maintained in
[`docs/install.md`](https://github.com/Overbrilliant/ob-1/blob/main/docs/install.md).

Supported launch paths:

- Homebrew: `brew install overbrilliant/tap/ob1`
- Native installer: `curl -fsSL https://github.com/Overbrilliant/ob-1/releases/latest/download/install.sh | sh`
- npm: `npm install -g @overbrilliant/ob1`
- Source checkout: `git clone https://github.com/Overbrilliant/ob-1.git`
