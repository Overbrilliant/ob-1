The canonical FreeLLMAPI guide is maintained in
[`docs/freellmapi.md`](https://github.com/Overbrilliant/ob-1/blob/main/docs/freellmapi.md).

OB-1 provisions FreeLLMAPI locally, can use anonymous bootstrap routes, and lets you add your own
provider keys for reliable capacity.
