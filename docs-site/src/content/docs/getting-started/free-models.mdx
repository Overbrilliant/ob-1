The canonical free-models guide is maintained in
[`docs/free-models.md`](https://github.com/Overbrilliant/ob-1/blob/main/docs/free-models.md).

OB-1 routes 150+ free models across 20+ cloud providers in-process — no separate service to run.
Keyless providers work with zero setup; adding free API keys to `~/.ob1/keys.env` activates more
providers automatically.
