OB-1 is a free open-source CLI coding agent with a local FreeLLMAPI first-run path, provider-neutral
OpenAI-compatible routing, persistent project memory, MCP, checkpoints, and multi-agent modes.

Canonical documentation source lives in [`CLI/docs/`](https://github.com/Overbrilliant/ob-1/tree/main/docs).
This Starlight site is the deployable shell for `docs.overbrilliant.com`.

## Start

```sh
brew install overbrilliant/tap/ob1
ob1
```

Choose **Start free**.
