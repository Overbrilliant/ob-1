# OB-1 Docs Site

Astro Starlight scaffold for `docs.overbrilliant.com`.

Canonical docs live in `../docs/`. The current site pages link back to those Markdown files so launch
content has one source of truth until a full docs migration is worth it.

```sh
npm install
npm run dev
npm run build
```

Deploy the static `dist/` output to the docs host.
