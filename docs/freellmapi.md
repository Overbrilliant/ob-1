# Free models (formerly FreeLLMAPI)

The self-hosted FreeLLMAPI proxy path has been replaced by an embedded free-models router built into
the CLI. There is no repository to clone, no local server to start, and no dashboard to sign into
anymore. Existing setups that previously configured FreeLLMAPI are automatically migrated to the
embedded router (model `auto`) on next launch — no action needed.

See the [Free models guide](free-models.md) for how it works, the keys file, and the `/free` command.
