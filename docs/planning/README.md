# Historical planning records

Dated build logs and design mockups kept as a record of how OB-1 was developed.
They describe the system as it was at the time of writing — including features
that have since been removed or reworked (e.g. the Council/Personas/fanout
modes deleted in July 2026 after they failed compute-matched evals). For the
current design, see [docs/multimind.md](../multimind.md) and
[docs/ARCHITECTURE.md](../ARCHITECTURE.md).
